## Overview

**Resource Toggle** is a lightweight, client-side quality-of-life mod that lets you instantly switch resource packs on and off with a simple keypress. Say goodbye to digging through Minecraft's sluggish settings menus just to enable your favorite packs!

## Features

* **One-Key Toggling:** Instantly enable or disable your selected resource packs with a single keypress while playing. No pauses, no menus!
* **Custom Keybinds:** Assign your preferred hotkeys to toggle specific packs (defaults to **Numpad 1-9** to toggle packs, and **Numpad 0** to clear all).
* **Simple Configuration:** Use a clean, user-friendly in-game config menu to select exactly which packs should be togglable.
* **Persistent Settings:** Your resource pack toggle configuration is automatically saved between game sessions.
* **Quick Search:** Find any pack instantly with the integrated search function.
* **Universal Compatibility:** Seamlessly works with all valid resource packs located in your `.minecraft/resourcepacks` folder.

## How to Use

1. Drop your resource packs into your `.minecraft\resourcepacks` directory.
2. Launch the game, open the in-game **Mods** menu, and navigate to the **Resource Toggle** config menu.
3. Select which resource packs you want to enable for quick-toggling.
4. Check the Minecraft Controls menu to view or change your keybinds (defaults to **Numpad 0-9**).
5. **Press your assigned keys anytime in-game to toggle the packs instantly!**

## Notes

* **Client-Side Only:** This mod operates completely on the client side. You can safely use it on multiplayer servers without affecting server gameplay or requiring the server to install it.
* **Loaders:** Completely compatible natively with both **Forge** and **NeoForge**.
