# Contributing to ResourceToggle

First off, thank you for considering contributing to ResourceToggle! It's people like you that make open-source mods such a great community.

## How to Contribute

1. **Fork the repository** to your own GitHub account.
2. **Clone the project** to your machine.
3. **Create a branch** locally with a succinct but descriptive name.
4. **Commit your changes** to the branch.
5. **Push your branch** to your fork.
6. **Open a Pull Request** in this repository.

## Setting up your Workspace

This project is a multi-loader mod setup containing both Forge and NeoForge.
1. Navigate to either the `Forge` or `NeoForge` directory.
2. Run `./gradlew genIntellijRuns` (or the equivalent for Eclipse/VSCode) to generate run configurations.
3. Use your IDE's run configuration to launch the Minecraft Client for testing.

## Reporting Bugs

If you find a bug, please create an issue on our GitHub repository. Include:
- Your Minecraft version
- Your Forge/NeoForge version
- The ResourceToggle mod version
- A detailed description of the issue and how to reproduce it
- A crash report or latest.log (if applicable)

## Suggesting Features

We welcome new features! Open an issue with the "enhancement" or "feature request" tag and explain how your feature would work and why it would be beneficial to the mod.
